ПОЛУЧЕНИЕ ОТ СЕРВЕРА (ВХОДЯЩИЕ КОМАНДЫ):
    СИСТЕМНЫЕ:
        SYS$SHUTDOWN - сервер выключается, ?: запомнить айпи и попытатся реконнектиться?
        SYS$DISCONNECT - сервер просит отключиться => отключиться от сервера
        SYS$MSG$<text> - пришло сообщение => вытащить текст и вывести?
    ЛОГИН:
        ОШИБКИ:
            id - id, с которым осуществлялся запорс на подключение
            LOGIN$INVALID_SYNTAX$<data>: неверный синтаксис запроса
                data - то, что получил сервер
            LOGIN$INVALID_ID$EXISTS$<id>: попытка зарегистрировать id, который уже зарегистрирован
                (ВОЗВРАЩАЕМЫЙ ID ПОЛОЖИТЕЛЬНЫЙ) => перегенирировать id, или зайти под существующим
            LOGIN$INVALID_ID$ONLINE$<id>: попытка логиниться с id, владелец которого уже онлайн
                => выдать ошибку
            LOGIN$INVALID_ID$FREE$<id>: попытка логиниться с id, который ещё не зарегистрирован
                => выдать ошибку
        УСПЕХ:
            LOGIN$CONNECT$<root>$<id> - вы подключились к серверу как root с айди id => всё ок
    КОМАНДЫ:
        НЕВЕРНЫЙ ВВОД:
            INVALID$DATA$<data> - неправильный синтаксис,
                data - то, что получил сервер => показать пользователю
            INVALID$SELF_ID$<id> - попытка отправки реквеста самому себе,
                id - свой id => показать пользователю
            INVALID$ADMIN_ID$<admin_id> - попытка отправки реквеста другому админу,
                id - id админа, которому пытались отправить реквест => показать пользователю
            INVALID$FREE$<id> - попытка отправки команды на несуществующий id,
                id - id, на который пытались отправить команду => показать пользователю
            INVALID$OFFLINE_CLIENT$<id> - попытка отправки на клиента,
                id - id, на который пытались отправить команду
            INVALID$OFFLINE_ADMIN$<id> - попытка отправки на клиента,
                id - id, на который пытались отправить команду
        КОМАНДА ОТ АДМИНА:
            <req_id>$<command>$<command_args> - команда с id req_id,
                командой command и параметрами command_args => выполнить команду и отправить результат
        РЕЗУЛЬТАТ ВЫПОЛНЕНИЯ ОТ КЛИЕНТА:
            <client_id>$<command>$<command_args>$<success> - клиент с id client_id выполнил команду
             command с параметрами command_args и получилось или не получилось => показать пользователю
    ИНФОРМАЦИЯ: (только для админов)
        ОШИБКИ:
            INFO$ERROR - неизвестная ошибка получения информации => показать пользователю
            INFO$ERROR$ACCESS_DENIED - попытка получения информации клиентом => хз, так делать не надо
            INFO$ERROR$INVALID_ID$<id> - ошибка получения ip адреса клиента id => показать пользователю
            INFO$ERROR$INVALID_SYNTAX$<request> - неверный синтаксис при попытке полученя информации,
                request - полученный сервером запрос
        УСПЕХ:
            INFO$ONLINE$<id_list> - список всех онлайн id (формат: [id1, id2, id3])
            INFO$REG$<id_list> - список всех зарегистрированных id (формат: [id1, id2, id3])
            INFO$ADMINS$<id_list> - список всех зарегистрированных админов
            INFO$CLIENTS$<id_list> - список всех зарегистрированных клиентов
            INFO$IP$<ip_address> - ip адрес клиента по id
ОТПРАВКА НА СЕРВЕР (ИСХОДЯЩИЕ КОМАНДЫ):
    ЛОГИН:
        РЕГИСТРАЦИЯ:
            <root>$id, где root = 'C' - клиент, 'A' - админ, id - число от -32000 до -1 включительно
        АВТОРИЗАЦИЯ:
            <root>$id, где root = 'C' - клиент, 'A' - админ, id - число от 1 до 32000 включительно
    ПОЛУЧЕНИЕ ИНФОРМАЦИИ (ТОЛЬКО АДМИН):
        A$INFO$ONLINE - список всех онлайн id
        A$INFO$REG - список всех зарегистрированных id
        A$INFO$ADMINS - список всех онлайн админов
        A$INFO$CLIENTS - список всех онлайн клиентов
        A$INFO$HEALTH - информация о использовании ресурсов сервером
    КОМАНДЫ:
        АДМИН:
            A$<client_id>$<command>$<command_args>:
                client_id - id клиента, который должен выполнить команду
                command - команда
                command_args - параметры команды
            A$GET_INFO$<id> - получение информации о id
        КЛИЕНТ:
            C$<command_id>$<client_id>:
                command_id - id команды, которую выполнил клиент
                client_id - id клиента, который выполнил команду
